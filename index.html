codeSpark

一个具有code review功能的web项目. 使用或者将会使用的技术包括：

  • 构建工具:yeoman
  • node.js开发框架:Express
  • css预处理器:less
  • Javascript 预处理器:coffee
  • Javascript MVW 开发框架: AngularJS
  • 模块化开发类库:require.js
  • 工具库:Underscore.js
  • ...

有些技术我也不是很熟悉,需要边学边实践.欢迎感兴趣的同学指教和贡献代码,共同维护这个项目.

Running

1.clone下整个项目

2.运行npm install 安装项目依赖（具体可以在package.json中具体查看）

3.运行grunt server
